I am **Álvaro Valencia Maiquez**, a Robotics Software Engineering student at URJC (Universidad Rey Juan Carlos, Madrid).

This blog documents my hands-on work in mobile robotics, service robotics, embedded systems, and robot modeling & simulation — mostly built with **ROS 2, Gazebo, Python and C++**.

![Álvaro Valencia](/assets/images/Without_random.png)

## Projects

- **Bartender robot**: In my second year, in the Software Architecture course, I worked with three classmates on a bartender robot project. Check out the project on [GitHub][Github-1].

- **Raspberry Pi**: In the Sensors and Actuators course, I did practical work with Raspberry Pi and have many example codes of its uses. Check out the project on [GitHub][Github-2].

- **ROS 2 Rover**: Modeling, simulation and control of an armed rover in ROS 2 (Blender, Gazebo, MoveIt, PyBullet). Repo on [GitHub][Github-3], write-up [here](/modelating-simulation/).
